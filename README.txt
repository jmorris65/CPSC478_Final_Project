Effects:
1) Soft Shadows
2) Depth of Field

Instructions To Build and Run:
1) Copy correct Makefile.* to Makefile (In the zoo use "cp Makefile.zoo Makefile")
2) Copy correct main.cpp.* to main.cpp (In the zoo use "cp main.cpp.zoo main.cpp")
3) Build the executable (Use "make")
4) Run the executable (Use ./run)
5) Output should be "final.0270.ppm"

6) If running on hpc cluster grace, submit buildRunMPI.sh as a batch job (Use "sbatch buildRunMPI.sh")
